## Design Principles

1. **Simplicity First** - One primary action per screen
2. **Large Touch Targets** - Minimum 48dp, check-in button 120dp+
3. **High Contrast** - Accessible for all users including elderly
4. **Calm Urgency** - Serious topic handled with care, not panic
5. **Trust Through Transparency** - Always show what will happen
